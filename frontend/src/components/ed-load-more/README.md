# EdLoadMore

* 下拉加载更多 组件

 
## 可选的配置项：


 配置项 | 类型 | 描述 | 默认值 
------------------|---------|-------------------------------- |-----
 value            | Boolean      | 是否在加载       | false
 loadingText      | String       | 加载状态文字     | vertical
 finishedText     | String       | 无更多数据文字    | true
 immediateCheck   | Boolean      | 是否立即加载      | 50
 offset           | Number       | 距离底部最小距离   | 300
